# Project Plan

## Project Name
Smart Billing SaaS

## Goal
Build a production-ready Java + React SaaS application to learn:
- Spring Boot
- Spring Security
- JWT Authentication
- PostgreSQL
- Redis
- Kafka
- React
- Tailwind CSS
- Docker
- Deployment

## Modules
1. Authentication & Authorization
2. Customer Management
3. Product / Service Management
4. Invoice Management
5. Expense Tracking
6. Dashboard & Reports
7. Redis Caching
8. Kafka Event System
9. Deployment

## Daily Learning Rule
- 10 min concept
- 40 min coding
- 10 min testing + commit
