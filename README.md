# Smart Billing SaaS

A production-style full-stack billing and expense management application built using:

## Backend
- Java
- Spring Boot
- Spring Security
- PostgreSQL
- Redis
- Kafka
- JWT

## Frontend
- React
- Vite
- Tailwind CSS
- Axios

## Features (Planned)
- User authentication & authorization
- Customer management
- Product/service management
- Invoice generation
- Expense tracking
- Dashboard analytics
- PDF export
- Redis caching
- Kafka event-driven notifications
- Dockerized deployment

## Project Goal
To build a deployable, monetizable SaaS project while learning modern backend and frontend development.
